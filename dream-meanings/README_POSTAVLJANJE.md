# Dream Meanings — програмски SEO сајт (постављање)

30 страница + hub, свака циља једну претрагу („dream about snakes meaning"…) и води на апликацију. Ради у позадини — Google их индексира и доносе USA/глобални саобраћај без твог рада.

## Постављање
1. У репоу **`dream`** направи фолдер **`dream-meanings`** (или само повуци цео овај фолдер).
   - Add file → Upload files → превуци **све** `.html` + `sitemap.xml` → Commit.
2. URL-ови постају:
   - Hub: **`kannacorp.github.io/dream/dream-meanings/`**
   - Странице: `…/dream/dream-meanings/snakes.html`, `…/falling.html`, итд.
3. **Sitemap:** отвори свој главни `sitemap.xml` (у root репоу) и додај ред:
   `<url><loc>https://kannacorp.github.io/dream/dream-meanings/</loc></url>`
   (или у Google Search Console додај и `…/dream-meanings/sitemap.xml` као засебан sitemap).
4. У **Google Search Console** (Dream property) → Sitemaps → додај `dream-meanings/sitemap.xml` → Submit. Затим URL Inspection → hub → Request Indexing.

## Линк ка апликацији
Свака страница има дугме „Get it free on Google Play" (EN апликација, `com.kanna.dreaminterpretation`, са `utm_source=seo`) + линк на landing `/dream/`.

## Како да прошириш (кад пожелиш)
- Ово су **најтраженијих 30** симбола. Речник у апликацији има хиљаду+. Пошаљи ми списак додатних симбола (или значења из апликације) → генеришем још 50–200 страница по истом шаблону. Више страница = више бесплатног саобраћаја.
- Ако хоћеш **тачан текст из апликације** уместо мог — пошаљи га, заменим значења (боље за конзистентност са апликацијом).

## Ситнице
- Значења су писана у бренд-тону „за самоспознају, не проricање" (важно за поверење и Google).
- Свака страница има унутрашње линкове ка сродним симболима (добро за SEO) + FAQ структурисане податке.
- Дизајн је ноћни/индиго као Dream страница — уклапа се.

> Реалан рок: SEO није инстант — прве Google посете за 2–6 недеља после индексирања, па расту. Али једном постављено, ради месецима без тебе.
